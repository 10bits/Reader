使用说明：

一、快捷键
1.  F12：     隐藏/显示边框
2.  Ctrl + O：打开新文件
3.  Ctrl + F: 全文关键字查找，并跳转到指定位置
4.  Ctrl + T：置顶/取消置顶
5.  Ctrl + G: 全文进度百分比跳转
6.  Ctrl + →：快速跳转到下一章
7.  Ctrl + ←：快速跳转到上一章
8.  →：       下一页
9.  ←：       上一页
10. ↑：       向上滚动 N 行 (N: setting > config > 文本滚动速度，默认为1)
11. ↓：       向下滚动 N 行 (N: setting > config > 文本滚动速度，默认为1)

二、热键
1.  Alt + H： 快速隐藏/显示窗口 (可以在 setting > config > 热键设置，进行自定义)

三、翻页
1.  鼠标左键单击：下一页
2.  鼠标右键单击：上一页
3.  鼠标向上滚动：上N行 (N: setting > config > 文本滚动速度，默认为1)
4.  鼠标向下滚动：下N行 (N: setting > config > 文本滚动速度，默认为1)

四、其他设置
1.  setting > font：   字体设置
2.  setting > color：  背景颜色设置
3.  setting > config： 进入设置菜单
4.  setting > default：还原默认设置

五、备注
1.  如果需要还原软件最原始的状态（或者清除File菜单里面的文件列表）：
    请删除Reader.exe同级目录下面的 “cache.dat” 文件。
    该文件为隐藏文件。
    至于如何显示隐藏文件，请自行百度。
2.  三种新文件打开方式：
    2.1 Ctrl + O
    2.2 菜单：File > Open
    2.3 文件拖拽
3.  边框隐藏时，鼠标左键按住页面顶部，可以进行拖动
